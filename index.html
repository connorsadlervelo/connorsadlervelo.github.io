Download CFS Postman Companion here

Installation and Usage
  1. download the zip above
  2. unpack the zip into a new empty folder
  3. run a Terminal and cd into that folder
  4. cd bin
  5. ./postmancompanion
    The application should now be running on port 18080 by default
  6. Make a request from Postman like this:

        http://localhost:18080/?showMessage=my message blah&promptFieldNames=field1&promptFieldNames=field2

    You must specify a "showMessage" parameter and one or more "promptFieldNames" parameters
  7. Fill in the values in the UI that pops up
    You'll see you will get a prompt box in the UI for every "promptFieldNames" parameter you specify

        {"field1":"hello","field2":"world"}
